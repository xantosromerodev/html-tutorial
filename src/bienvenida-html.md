## Fácil de aprender con "Prueba tú mismo"
Con nuestro editor "Prueba tu mismo", tú puedes editar el código HTML y ver el resultado.

**Ejemplo**
```html
<!DOCTYPE html>
<html>
<head>
<title>Título Página</title>
</head>
<body>

<h1>Este es un encabezado</h1>
<p>Este es un párrafo.</p>

</body>
</html>
```
